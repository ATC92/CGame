Detailed Description

RenderData, Struct that holds all the data for rendering the map.

This struct contains all the necessary data for rendering the map in the game. It includes a pointer to the map data, a pointer to the tile map, an enum to select the current map, and an array of textures used for rendering.

mapsData Pointer to the map data \ tileMap Pointer to the tile data \ mapSlctr Enum that selects the current map \ texturesArray Array of textures used for rendering the map \ emptyTexture Empty texture used for rendering empty tiles
