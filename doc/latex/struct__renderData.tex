\doxysubsection{Detailed Description}
Render\+Data, Struct that holds all the data for rendering the map. 

This struct contains all the necessary data for rendering the map in the game. It includes a pointer to the map data, a pointer to the tile map, an enum to select the current map, and an array of textures used for rendering.
